# WASM examples written in Rust

## Build

1. Install the WASM toolchain for Rust:
```sh
rustup target add wasm32v1-none
```
You can also experiment with the standard `wasm32-unknown-unknown` toolchain,
but its feature set is unstable and in the future it might extend beyond
what WALC supports.
1. Run the make command. This will install all the `.wasm` files into the `bin`
   directory.
```sh
make
```


## Run

Example:

```sh
walc bin/mandelbrot.wasm -o target/mandelbrot.walc
../interpreter/lamba.ts target/mandelbrot.walc
```
